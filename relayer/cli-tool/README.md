relayer
=======

Sender - Receiver

USAGE
```
  $ relayer [PARAM]
```
OPTIONS
```
  -d, --decode      Decrypt file
  -g, --gen         Get new key
  -h, --help        show CLI help
  -n, --getName     Read Author list
  -r, --rec[ip]     Receive file
  -s, --send[file]  Send file with local Public key
  -v, --version     show CLI version
```
